# Model card — TF-IDF Logistic Regression synthetic baseline

## Status

Evaluation-only; **not promoted or deployed as an educational classifier**.

## Training data

384 original, unreviewed synthetic questions generated by LearnStep. Fixed grouped split: train 264, validation 56, test 64; zero `template_group_id` overlap. Dataset SHA-256: `8a857298e82209d32c2415c57b4364247f4822ae890e9ada1c5e44c58da48814`.

## Model

Feature union of word TF-IDF (1–2 grams) and character TF-IDF (3–5 grams), followed by class-balanced Logistic Regression with seed `20260713` and `max_iter=1000`. Separate evaluations were run for cognitive level and difficulty.

## Recorded synthetic results

- Cognitive level: macro/weighted F1 1.00 on synthetic grouped test n=64; majority macro F1 0.0556.
- Difficulty: macro/weighted F1 1.00 on the same synthetic test; majority macro F1 0.1818.
- Per-class results, confusion matrices, CPU latency, and split details: `artifacts/evaluation/classifier_metrics.json`.

## Limitations and decision

Phrasing contains generator cues and is not representative of reviewed classroom questions. There is no human labeling/adjudication, original-only slice, 100-error human analysis, calibration study, or real class/subject slice evidence. No serialized model is committed and no classifier is promoted. DistilBERT was not trained because a comparison on this corpus would be misleading.
